# AURA Canonical Architecture

## Core Principles

1. One responsibility per domain
2. One canonical engine per capability
3. No duplicated cognition systems
4. Experimental systems belong in lab/
5. core/ orchestrates but does not own all logic
6. engine/ is transitional and gradually deprecated
