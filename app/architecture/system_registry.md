# AURA SYSTEM REGISTRY

---

# PURPOSE

This document defines the official classification of all systems inside AURA.

Every folder, engine, and subsystem must belong to one of the following categories:

- ACTIVE
- SUPPORT
- RESEARCH
- FUTURE
- LEGACY
- DUPLICATE

This registry acts as:
- architecture governance
- consolidation tracking
- integration planning
- cleanup coordination
- future system mapping
